# Python常用模组

## 概述

此为个人常用模组，模组化之后利于日常开发加速.

## 使用

1. 下载模组后，放入同个专案中，以mail.py为例
2. 引用模组
	`from mail import *`
3. 调用模组函数

## 环境

* Python 3.7.x
* MAC OS 10.11.X

## 模组清单

* [发送邮件](./mail.py) - 发送邮件（可添加附件）
* [WIFI名称](./wifi.py) - 获取WIFI名称（MAC OS ONLY）
* [执行SHELL指令（非变数)](./shell.py) - 执行SHELL指令，结果不可作为变数
* [执行SHELL指令（变数)](./shellreturn.py) - 执行SHELL指令，结果可作为变数
